BackEnd

For the Backend was used Entity Framework for the communication between the database and the service, .NET Core 5 for the WebAPI, JWT as a security validator.

The database used was SQL Server hosted on Azure Cloud.

FrontEnd

For the frontend the app was built with Angular, HTML5; CSS3 and Bootstrap for the UI/UX. The web site splits in components and use a single Service for the communication with the REST Web Service.

It's a single project but validate if user is logged in and shows the corresponding resources.

DEV Environment

Backend

The following tools are required to run the app:.

  • Visual Studio 2019

Open the project in Visual Studio and clic on the run button.

Frontend

The following tools are required to run the app:.

  • Nodejs
  • npm
  • Angular
  • Angular CLI

In a command prompt follow this instructions:

  1. Navigate to the workspace folder JSSTORE.
  2. Run the following command to install the node_modules folder ng serve -o.
  3. Run the following command ng serve -o.
